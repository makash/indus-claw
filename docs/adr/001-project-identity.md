# ADR-001: Project Identity and Positioning

## Status
Accepted

## Context
We are building an India-centric fork of OpenClaw — an open-source personal AI agent framework. The project needs a clear identity, domain, and positioning that reflects its Indian roots while maintaining developer appeal.

## Decision

- **Name**: IndusClaw
- **Domain**: indusclaw.in
- **Positioning**: India-centric OpenClaw fork, developer-first
- **Tagline**: TBD (options: "India's Open AI Agent" / "Open-Source AI Agents, Made in India")
- **Tone**: Developer-first, technical, approachable
- **Visual Identity**: Indus Valley / Harappan civilization inspired, minimalist, earthy colors

## Consequences
- All branding, documentation, and marketing will use "IndusClaw" as the project name
- The `.in` TLD signals Indian origin and focus
- Developer-first tone means technical accuracy over marketing fluff
- Harappan visual motifs provide a unique, culturally rooted identity distinct from OpenClaw
